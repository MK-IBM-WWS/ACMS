У меня еще есть вот такое окно редактирования информации о сотрудниках той же таблицы, необходимо при открытии окна для этого необходимо сразу при открытии окна подключиться к бд по данным из конфига и передаваемому логину и паролю, если не передан id загружаемой записи, то загрузить из бд самую первую существующую запись, иначе загрузить запись по переданному id из таблицы staff в поля: В поле lefio вносится фио сотрудника, в поле lephone вносится номер телефона сотрудника, в комбобокс cbiswork вносится булевое значение True/Fase, при это если значение истина должно отображаться "Работает", иначе "Уволен", в поле leposition вносится название должности сотрудника, в поле ledepo вносится название отдела, в котором работает сотрудник и адрес отдела чеерез разделитель | в процессе ввода должна выводиться подсказка повводу названия отдела с возможностью автодополнения, потом по внесенным данным выводится id отдела. По нажатию на btndelete необходимо удалять текущую открытую запись из таблицы staff (предварительно запросив подтверждение с фио удаляемого пользователя). по нажатию на btnsave необходимо обновлять текущую открытую запись таблицы staff значениями из ранее перечисленных полей. По нажатию на btnback переходи к предыдущей записи таблицы departments и отображай ее данные, если она есть, если нет оставайся на той же записи, соответственно по нажатию на btnforward необходимо перейти к следующей записи, если она есть. По нажатию на btnupdate необходимо перезапросить данные о текущей записи из таблицы staff. lesearch - сюда вводится фио сотрудника и его телефон через разделитель | для поиска, необходимо показывать подсказку ввода c возможностью автодополнения, по нажатию на btnsearch необходимо открыть запись с введенным в это фио и телефоном, если она есть, иначе выдать ошибку. После нажатия на кнопку btnlogout открывается (если не открыто) окно HrWindow(QString &login, QString &password, QWidget *parent = nullptr); куда передаются значения логина и пароля подключения к бд, и закрывается окно редактирования сотрудника.
